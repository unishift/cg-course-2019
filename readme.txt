Сторонние библиотеки
-------------------------------------------
Необходима библиотека DevIL для загрузки изображений
Команды для Debian/Ubuntu:
    sudo apt install libdevil-dev


Управление
-------------------------------------------
Закрыть окно - Escape

Движение - WASD + R/F + Q/E + мышь

Ускорение - Shift

Время (on/off) - Space
    Отвечает за движение объектов

Вернуться к стандартному виду - 0
    Сбрасывает все настройки и возвращает камеру на исходное положение

Мягкие тени (off/on) - 1
    Отлично видны на "платформе"

Отражения (off/on) - 2
    Лучше всего видно на маленькой зеркальной сфере
    Так же хорошо видно на "платформе"

Преломления (off/on) - 3
    Сфера в середине сцены и куб слева

Anti-aliasing (off/on) - 4
    Хорошо видно на тенях на "платформе" (c выключенными мягкими тенями)

Ambient Occlusion (off/on) - 5
    Хорошо видно на фрактале
